# 💼 Landing Page

Учебный pet‑project — адаптивный лендинг для сервиса поставок лифтового оборудования **SPARTSPRO**.  
Проект демонстрирует навыки в Pixel Perfect, работе с чистыми **HTML** и **CSS**, построении карточек товаров и информационных секций, а также использовании BEM‑подобной методологии именования классов.

---

## 🚀 Технологии

- HTML5  
- CSS3  
- Flexbox  
- CSS Grid  
- Google Fonts  
- BEM‑подобная структура классов  
- Vanilla JS 

---

## ✨ Что реализовано

- Фиксированный header с навигацией и логотипом SPARTSPRO  
- Адаптивное бургер‑меню:
  - переключение состояния (иконка «гамбургер» → «крестик»)
  - закрытие по клику на пункт меню
  - закрытие по клику вне области меню
  - управление доступностью через `aria-expanded`
- Секция «Рекомендуемое оборудование»:
  - карточки товаров с изображением, названием и статусом наличия
  - визуальное разделение статусов:
    - зелёная метка — товар в наличии
    - красная метка — товар под заказ
  - адаптивная сетка карточек:
    - 2 колонки на десктопе
    - 1 колонка на планшете
    - 2 компактные колонки на мобильных
- Секция «О компании»:
  - комбинированный блок «картинка + текст» с описанием компании
  - список преимуществ с кастомными маркерами (иконки через `background-image`)
  - call‑to‑action кнопка «Подробнее»
- Адаптивная и аккуратная сетка:
  - несколько брейкпоинтов (`1440px`, `1160px`, `768px`, `576px`, `390px`)
  - изменение размеров текста, отступов и структуры блоков
- Работа с CSS‑переменными:
  - цвета, шрифты, размеры контейнера задаются в `:root`
  - удобная поддержка и масштабирование стилей
